# Wishlist API

A simple NodeJS API build using
- Express
- MongoDB
- Docker

# Setup

## Installation
```
git clone  [...] wishlist-api
cd ./wishlist-api
npm install
```

## Running
To run the application type in terminal

```
docker-compose up
```

## Testing
To run the tests in live-mode, type:
```
npm run test
```

To run all tests at once:
```
npm run test:once
```
